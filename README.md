# CHI-Vehicle-Tracking
This project is based on YOLOv4 and able to detect and track the vehicles in a video.

## Env: 
  * OpenCV 4.6.0

  * Python 3.8.16

  * Numpy 1.23.5

## Steps to run

  * Run test.py file

  * Remember to revise the filepath and add your own video
